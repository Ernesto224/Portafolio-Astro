# AGENTS.md

## Mision del proyecto
Objetivo: Desarrollar un Portafolio Digital Interactivo de marca personal.
Estilo: Moderno y Minimalista, con énfasis en la limpieza visual y la usabilidad.
Inspiración Estética: Identidad visual y experiencia de usuario inspirada directamente en el menu BIOS (Basic Input/Output System) de la consola PlayStation 2 (PS2) 

## Setup commands
- Install dependencies: `pnpm install`
- Start development server: `pnpm dev`
- Run test suite: `pnpm test`

## Code style
- Sintaxis: Uso estricto de TypeScript.
- Terminación de Sentencia: Obligatorio el uso de punto y coma (`;`) al final de cada sentencia.
- Funciones: Uso estricto de funciones de flecha (Arrow Functions).
- Nomenclatura: Todas las variables, funciones y métodos deben nombrarse en `camelCase`.
- Principios de Diseño: Aplicar el Principio de Responsabilidad Única (SRP) en todos los métodos y componentes.
- Interfaces: Nombres de interfaces y tipos en singular (e.g., `User`).
- Nombres de Archivos: Los archivos de componentes o módulos principales deben usar PascalCase (e.g., `HeaderComponent.js`).
